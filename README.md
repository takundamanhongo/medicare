# 🏥 MediCare — Hospital Management System

A full-stack hospital information system built with Python/Flask
and SQLite as a capstone project for Database Design \& Development
(HDSC213) at the University of Zimbabwe.

## Features

* Patient registration and records management
* Appointment scheduling
* Role-Based Access Control (RBAC) — Admin, Doctor, Receptionist
* Complex SQL reporting and queries
* ER Diagram and full relational schema

## Technologies Used

* Python 3 \& Flask
* SQLite Database
* HTML/CSS Templates
* SQL Queries \& Reporting

## How to Run

1. Clone the repository:
git clone https://github.com/takundamanhongo/medicare
2. Install dependencies:
pip install -r requirements.txt
3. Run the application:
python hospital\_web\_final.py
4. Open your browser at:
http://localhost:5000
